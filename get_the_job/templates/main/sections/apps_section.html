Apps & APIs

A few of my open source web apps, APIs, and console apps/scripts are listed below with links to try them out. My largest, most recent, and featured side project is Snypit.

Featured

Snypit is a modern, code snippet management system built with Flask. Snypit enables registered users to quickly save and retrieve code snippets for their projects, while offering the benefits of language-specific syntax highlighting and over 50 popular code editor themes.

Technologies

  • Python

  • Flask

  • JavaScript

  • AJAX

  • Materialize

  • MySQL

  • SQLAlchemy & PyMySQL

  • CodeMirror

  • Regular Expressions

  • API Consumption

  • Password Hashing

  • Security Tokens

  • CSRF Protection

  • Session Management

  • Web Assets/Minification

  • Anaconda/Pip

Features/Capabilities

  • 50+ Code Editor Themes

  • Pinned Snippets

  • Popular Snippets

  • Create, Read, Update, Delete Snippets

  • Easy-Access Side Bar

  • Search Functionality

  • Account Creation

  • Recent Snippets

  • Snippet Quick Copy

  • Email Confirmation/Forgot Password

  • Syntax Highlighting

  • Update Account Settings

Web Apps

VZTabs

VZTabs allows you to create modern, custom, and clutter-free start and new tab pages. Complete with Google Search functionality, VZTabs displays the time, date, local weather, and a warm greeting message.

 Launch
RPG

RPG is a web application used to create strong, secure passwords. RPG leverages the Python secrets module to generte cryptographically strong, random strings suitable for passwords, user authentication, and security tokens.

 Launch
Ports

A modern solution for determining a port status of a web server. Enter the URL or IP address of the server and the port to scan. Ports leverages the power of the network mapping utility, Nmap, to scan and determine the target port's status.

 Launch
GetIP

A quick and easy solution for retrieving IP addresses of online servers. GetIP uses the Linux network adminstration tool, Dig (Domain Information Groper), to query DNS servers and fetch server IP addresses connected to domain names.

 Launch
C3

A quick and easy solution for converting hexidecimal color codes to RGB or RGB color codes to hexidecimal. C3 is an invaluable tool for web developers or graphic designers.

 Launch
Donations

Donations is a web app used for my open-source projects to collect donations from supportive application users. It features two payment methods, PayPal and Credit Cards via Stripe.

 Launch

APIs

GeoAPI

GeoAPI is a RESTful API that provides geolocation functionality to independent applications. The API returns geodata linked to an IP address by leveraging the MaxMind GeoLite2 database.

 Launch
Datetime-Utils-API

Datetime-Utils-API is a RESTful API that provides datetime formatting, timezone conversion, and datetime calculation services to independent applications.

 Launch
VZLink

VZLink is an API & web application used to create short, memorable links to replace those long, and often impossible to remember, links to your favorite places on the web.

 Launch

Console Apps/Scripts

Web Server Scanner

A Python console application that monitors a list of web servers. The application will send notifications if a web server returns an undesirable response code.

 GitHub
AWS EC2 Manager

A Python console script that enables AWS users to manage their EC2 instances conveniently from any terminal.

 GitHub
Password Manager

A Python console application used to securely manage passwords. The password manager leverages SQLite and PGP encryption for a perfect balance of performance and security.
